# Sentiment Analysis – MLOps Production Pipeline

## Overview
This project demonstrates an end-to-end MLOps workflow for deploying a sentiment analysis model using MLflow, FastAPI, Docker, and Kubernetes.

## Features
- MLflow experiment tracking and model versioning
- REST API inference service using FastAPI
- Docker containerization
- Kubernetes deployment configuration
- Production-style model lifecycle management

## Tech Stack
- Python
- Scikit-learn
- MLflow
- FastAPI
- Docker
- Kubernetes
